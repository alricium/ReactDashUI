# React Dashboard Sidebar & Navbar

Bu proje, React ile geliştirilmiş responsive bir **Sidebar ve Navbar** arayüzünü içerir. Sidebar açılır/kapanır, dropdown menüler çalışır ve aktif linkler vurgulanır. Navbar ise bildirim ve profil alanlarını içerir.

This project includes a responsive **Sidebar and Navbar** interface developed with React. The sidebar opens/closes, dropdown menus work, and active links are highlighted. The navbar contains notification and profile areas.

---

## Özellikler & Features

- Responsive Sidebar (mobil ve masaüstü)
- Dropdown menüler ve alt linkler (Dropdown menus and sublinks)
- Aktif menü vurgulama (Active menu highlighting)
- Hamburger menu & X iconu mobil cihazlar için
- Navbar üzerinde bildirim ve profil alanları
- Kolay özelleştirilebilir CSS değişkenleri (`:root` ile renkler ve radius)

---

## Teknolojiler & Technologies

- React (Function Components & Hooks)
- React Icons (`react-icons/ai`, `react-icons/io`, `react-icons/io5`)
- CSS / Flexbox
- Responsive Tasarım (Media Queries)
